# CSV Studio

**Smart CSV toolkit for Windows.**
Detect, clean, edit, and export CSV files with a few clicks — 100% offline.

\![version](https://img.shields.io/badge/version-1.0.0-blue)
\![platform](https://img.shields.io/badge/platform-Windows%2010%2F11-blue)
\![license](https://img.shields.io/badge/license-proprietary-lightgrey)

## Features

- 🔍 **Auto-detection** of encoding, separator, quote character, and headers
- 📊 **Column statistics** (type, empty values, distinct, duplicates, min/max)
- ✏️ **Inline editing** — edit cells directly, add / duplicate / delete rows and columns, and rename column headers
- ↩️ **Undo / Redo** with `Ctrl+Z` / `Ctrl+Y`
- 🛠️ **Cleanup operations** — trim spaces, remove empty rows, remove duplicate rows, remove duplicate columns, change case (UPPER / lower / Title)
- 🔎 **Find & Replace** across all cells, with regex and match-case options
- 📦 **Batch processing** — clean dozens of files at once
- 💾 **Export** to CSV (with custom encoding/separator) or JSON
- 🎨 **Light / Dark theme**, with the modern Mica backdrop on Windows 11
- 🔒 **100% offline** — your data never leaves your computer

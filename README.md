# CSV Studio

**Smart CSV toolkit for Windows.**  
Detect, clean, and export CSV files with a few clicks — 100% offline.

## Features

- 🔍 **Auto-detection** of encoding, separator, quote character, and headers
- 📊 **Column statistics** (type, empty values, distinct, duplicates, min/max)
- 🛠️ **7 cleanup operations**: trim, remove empty rows, remove duplicates, change case, and more
- 📦 **Batch processing**: clean dozens of files at once
- 💾 **Export** to CSV (with custom encoding/separator) or JSON
- 🌙 Light / Dark theme
- 🔒 **100% offline** — your data never leaves your computer
